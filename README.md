# 😂 JokeApp

Aplicación sencilla para visualizar chistes consumidos desde una API externa y guardarlos como favoritos.

---

## 🧠 Descripción

**JokeApp** es una aplicación ligera cuyo objetivo principal es demostrar el consumo de APIs y la gestión básica de datos en una app Android.

Permite obtener chistes desde una fuente externa y guardarlos localmente para consultarlos más tarde.

## 🧱 Arquitectura

Aplicación estructurada siguiendo buenas prácticas:

* Separación de lógica y UI
* Uso de `ViewModel` para gestión de estado

## 🎯 Objetivo del proyecto

Este proyecto tiene un enfoque práctico y educativo:

* Demostrar consumo de APIs
* Manejo de datos remotos
* Persistencia básica de información (favoritos)
